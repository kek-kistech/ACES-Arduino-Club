# 📚 Recommended Arduino Books & Resources

This file lists books and tutorials that are useful for learning Arduino.  
Please note: **we only share summaries and links here**. Full copyrighted books should not be uploaded directly.

---

## 1. Arduino Projects Book (2012)
- **Authors**: Scott Fitzgerald, Michael Shiloh  
- **License**: Creative Commons Attribution-NonCommercial-ShareAlike 3.0  
- **Summary**: Hands‑on projects from the official Arduino Starter Kit (LEDs, sensors, motors, interactive designs).  
- **Link**: [Arduino Starter Kit Resources](https://arduino.cc/starterkit)

---

## 2. Funduino Arduino Tutorials (2016)
- **Publisher**: Funduino Service Team  
- **Summary**: Beginner‑friendly tutorials with practical tasks (blinking LEDs, sensors, servo control, LCDs).  
- **Link**: [Funduino Tutorials](https://funduinoshop.com)

---

## 3. Sams Teach Yourself Arduino Programming in 24 Hours (2015)
- **Author**: Richard Blum  
- **Summary**: Structured guide to Arduino programming, focusing on C basics, libraries, sensors, motors, and networking.  
- **Note**: Copyrighted — please purchase or access legally.  
- **Link**: [Publisher Page](https://www.informit.com/store/sams-teach-yourself-arduino-programming-in-24-hours-9780672337123)

---

## 4. Microcontroller Programming with Arduino and Python (2021)
- **Authors**: Sudhakar Kumar, Manas Ranjan Das, Rajesh Kushalkar, Nirmala Venkat, Chandrashekhar Gourshete, Kannan M. Moudgalya  
- **License**: Creative Commons Attribution-NonCommercial-NoDerivatives (CC BY-NC-ND)  
- **Summary**: Explains Arduino programming using Python alongside the Arduino IDE. Covers LEDs, pushbuttons, LDRs, potentiometers, thermistors, servo motors, DC motors, and Modbus protocol.  
- **Link**: [FOSSEE Arduino Resources](https://floss-arduino.fossee.in)

---

## 5. Arduino Cookbook (O’Reilly, 2nd Edition)
- **Author**: Michael Margolis  
- **Summary**: A practical recipe‑style guide for solving common Arduino problems (hardware interfacing, sensors, communication, advanced projects).  
- **Note**: Copyrighted — please purchase or access legally.  
- **Link**: [O’Reilly Arduino Cookbook](https://www.oreilly.com/library/view/arduino-cookbook-2nd/9781449313876/)

---

## 📖 How to Use These Resources
- Start with the **Arduino Projects Book** for hands‑on basics.  
- Use **Funduino Tutorials** for classroom‑style exercises.  
- Dive into **Sams Teach Yourself Arduino Programming** for deeper programming knowledge.  
- Explore **Microcontroller Programming with Arduino and Python** if you want to combine Arduino with Python scripting.  
- Reference the **Arduino Cookbook** for troubleshooting and advanced techniques.
